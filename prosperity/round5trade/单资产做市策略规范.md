## 2. 已定义变量

`fair_i`：每个资产的 fair price，由上游模块提供。

---

## 3. 做市模块（quote）

### 3.1 规则

每个资产 i：

- bid_price = best_bid_i + 1
- ask_price = best_ask_i - 1
- bid_size = 10 - q_i
- ask_size = 10 + q_i

**过滤**：

- bid_price ≥ fair_i → bid_size = 0
- ask_price ≤ fair_i → ask_size = 0

### 3.2 伪代码

```python
POS_LIMIT = 10

def quote(q, best_bid, best_ask, fair):
    bid_price = best_bid + 1
    bid_size = POS_LIMIT - q
    if bid_price >= fair:
        bid_size = 0

    ask_price = best_ask - 1
    ask_size = POS_LIMIT + q
    if ask_price <= fair:
        ask_size = 0

    return {
        'bid_price': bid_price, 'bid_size': bid_size,
        'ask_price': ask_price, 'ask_size': ask_size,
    }
```

## 4. 对冲模块（take）

### 4.1 规则

内层订单（价格 == fair）出现时，take 至持仓归零。

- q > 0：take 内层 bid，卖出 min(q, v)
- q < 0：take 内层 ask，买入 min(|q|, v)
- q = 0：不操作

v 为内层订单可成交量。

### 4.2 伪代码

```python
def take(q, v, side):
    """
    side = +1: 内层为卖单（我方买入）
    side = -1: 内层为买单（我方卖出）
    返回: delta (正=买入, 负=卖出)
    """
    if side == -1 and q > 0:
        return -min(q, v)
    if side == +1 and q < 0:
        return min(-q, v)
    return 0
```

---

## 5. 调用顺序

每个 tick，对每个资产：

1. 若内层订单存在，调用 `take` 并下发 delta
2. 撤旧外层挂单
3. 调用 `quote` 下发新挂单
